# Arkanoid Agent

Mobile-first Arkanoid-style brick breaker with neon cyberpunk vibes. Built as a single-page HTML5 Canvas app.

## Run locally

Open `index.html` in a browser.

## Controls

- **Mobile:** drag/swipe in the lower area of the screen
- **Desktop:** Left/Right arrow keys
- **Optional:** move mouse near bottom area

## Deploy

Static hosting ready (GitHub Pages, Netlify, etc.).
